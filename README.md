# AI SEO Automation Tool

## 📌 Overview
AI-powered API that generates SEO content from keywords.

## 🚀 Features
- Generate SEO title
- Generate meta description
- Generate article outline
- Store generated results

## 🛠 Tech Stack
- FastAPI
- OpenAI API
- Python

## ⚡ Getting Started

```bash
pip install -r requirements.txt
uvicorn app.main:app --reload
```
